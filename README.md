To run this app,
On Backend, run mongod (MongoDB)

On FE:
first create a virtual environment:
```python -m venv venv ```

To actiave your venv, run 
```venv\Scripts\activate ```

To install all current requirements
```pip install -r requirements.txt``` 

To run program,
```python run.py```
